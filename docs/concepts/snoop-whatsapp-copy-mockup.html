Conceito para copiar a lista do dia num padrão que o WhatsApp entende: título em negrito, itens com bolinha, texto introdutório opcional. Mockup antes de implementar — nada disso existe no app ainda.

Seu fluxo hoje

  • Abre o projeto SNOOP → seção SALDOS DIÁRIOS
  • Entra em cada tarefa/subtarefa do dia
  • Seleciona a descrição manualmente e cola no WhatsApp
  • Formata “no olho” — negrito, bullets, data — toda vez

Formatação WhatsApp

  • *texto* → negrito
  • _texto_ → itálico (observações)
  • Quebra de linha + • no início → lista visual
  • O app não precisa gravar asteriscos na descrição — gera na hora de copiar

Lógica proposta (como implementaria)

1

De onde vêm os itens

Tarefa-pai do dia (ex. 15 Jul — Saldos) + subtarefas pendentes, ou todas as tarefas da seção SALDOS DIÁRIOS com vencimento = hoje. Título → negrito; descrição → texto após o travessão.

2

Template da mensagem

Cabeçalho fixo + lista gerada + rodapé opcional (salvo nas preferências do projeto SNOOP). Marcadores WhatsApp: * e _ só na hora de copiar — o banco continua texto limpo.

3

Onde fica no app

Chip WhatsApp no header da seção (só seções marcadas como “rotina diária”) ou menu ⋮ na tarefa-pai do dia. Sheet com preview + Copiar + toast “Copiado”.

4

Escopo mínimo (v1)

Sem editar template complexo: intro/rodapé padrão editável uma vez em Configurações do projeto. Não muda como você escreve descrições hoje — só automatiza o “montar mensagem”.
