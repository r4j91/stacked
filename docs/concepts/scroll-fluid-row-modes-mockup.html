Etapa 2 — modos mais fluidos (mockup)

Comparação visual do que existe hoje com duas propostas mais baratas no scroll. Nada disso troca o que você já tem: entram como opções extras no seletor de exibição do projeto (e, se quiser depois, como default em Hoje / Inbox).

Ideia: manter Balões, Balões+ e Lista. Somar Balões light (mesmo vibe de card, compositing mais leve) e Lista premium (flat Things, ainda mais limpa). Você escolhe qual usar — não é replace forçado.
